PROGRAM: InstallSimple
VERSION: 3.5 (January 8, 2024)
STATUS: FREEWARE

DESCRIPTION:
InstallSimple is a small utility to create installation packages
in seconds! Setups are packed into single exe-file. The extractor
module has extremely small size (only 24 KB overhead over compressed
data size). Using a very practical and intuitive assistant, you can
establish parameters for the installation process of your product
for any Windows platform.

SYSTEM REQUIREMENTS:
1. Operating Systems: Windows (95 and newer)

INSTALLATION:
To install InstallSimple simply unzip install-simple.zip
and run Setup.exe
